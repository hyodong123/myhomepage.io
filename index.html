Java

객체지향적인 프로그래밍 언어로, 다양한 분야에서 활용되며 크로스 플랫폼을 지원합니다. 우리는 Eclipse를 통해 자바를 프로그래밍 하기 위한 문법을 익히고 있습니다.

Algorithm

문제를 해결하기 위한 절차나 방법을 연구하는 학문으로, 효율적인 알고리즘을 설계하는 능력을 키우며, 백준 문제집을 통해 알고리즘을 학습하고 있습니다.

Interface Design

Visual Studio 툴을 이용하여 Html, CSS, JavaScript를 통해 웹페이지를 구현하며, 각 요소의 역할을 배웠으며, 웹페이지를 제작하는 코딩을 배웁니다.

데이터 베이스

SQL Developer는 Oracle에서 개발한 데이터베이스 관리 및 개발 도구입니다. SQL Developer를 사용하여 Oracle 데이터베이스를 관리하고, SQL 쿼리를 코딩하는 것을 배우고 있습니다.

기초 통계

R은 통계 분석 및 데이터 시각화를 위한 프로그래밍 언어입니다. R의 내장 함수와 원시데이터를 통해 우리는 많은 데이터들을 시각화하거나 테이블화 하여 데이터를 분석, 응용 하는 방법을 배우고 있습니다.

산업 안전

정보 보안은 컴퓨터 시스템 및 네트워크에서 데이터를 보호하는 프로세스와 기술을 포함하는 분야입니다. 주요 목표는 기밀성, 무결성, 가용성을 유지하고 데이터 및 시스템을 외부 위협으로부터 보호하는 것입니다.
